mysql配置

创建一个用户

debezium 配置

  • 下载 conflution官网下载社区版 tar.gz
    https://www.confluent.io/download/
  • 下载第三方 connector
    http://debezium.io/docs/install/
    解压，放入在share/java文件中

配置文件
debezium.properties

standalone.properties

  • 命令行启动
    启动：
    ./bin/connect-standalone -daemon ./etc/standalone.properties ./etc/debezium.properties

监听：
./bin/kafka-console-consumer –topic debeziym.touc_trade.t_reservation –zookeeper xx.96.160.128:2181

  • 网络命令

    https://docs.confluent.io/current/connect/managing.html
    https://docs.confluent.io/current/connect/restapi.html#connect-userguide-rest

查看
curl localhost:8083/connectors
curl localhost:8083/connectors/debeziym-connector-touc_trade-03/status | jq
查看配置
curl localhost:8083/connectors/debeziym-connector-touc_trade-03

创建
curl localhost:8083/connectors

删除
curl -X DELETE localhost:8083/connectors/debeziym-connector-touc_trade

重启
curl -X POST localhost:8083/connectors/debeziym-connector-touc_trade-03/restart
暂停
curl -X PUT localhost:8083/connectors/debeziym-connector-touc_trade-03/pause
恢复
curl -X PUT localhost:8083/connectors/debeziym-connector-touc_trade-03/resume

分布式启动

./bin/schema-registry-start -daemon ./etc/schema-registry.properties

./bin/connect-distributed -daemon ./etc/distributed.properties

创建
curl -X POST -H “Content-Type: application/json” –data ‘{“name”:”debeziym-connector-touc_trade-01”,”config”:{“connector.class”:”io.debezium.connector.mysql.MySqlConnector”,”database.user”:”debezium”,”database.server.id”:”1”,”tasks.max”:”10”,”database.history.kafka.bootstrap.servers”:”47.96.160.128:9092”,”database.history.kafka.topic”:”MySqlConnectorTotalTopic”,”database.server.name”:”debeziym”,”include.schema.changes”:”true”,”database.port”:”3306”,”database.hostname”:”47.96.160.128”,”database.password”:”Ku1LsgVs5XLN”,”name”:”debeziym-connector-touc_trade-03”,”database.whitelist”:”touc_trade”,”snapshot.mode”:”schema_only”}}’ localhost:8083/connectors
