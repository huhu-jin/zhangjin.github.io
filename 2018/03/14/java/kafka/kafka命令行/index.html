kafka

kafka 常用命令行

查看消费者组:

bin/kafka-consumer-groups.sh –bootstrap-server 192.168.137.51:9092 –list

其中test是默认的消费者组

console-consumer 是命令行控制台 的消费者组.每次新启动一个命令行消费者,同时会创建一个消费者组.所以命令行消费者不会竞争消费.

具体消费情况

./kafka-consumer-groups.sh –bootstrap-server 192.168.137.51:9092 –describe –group test

partition:分区编号

current-offset:当前偏置

log-end-offset:总的数据

lag:落后的个数

分区和偏置都是0开始

去除–excute 只是打印

–to-datetime 多少时间之前

–by-duration 多少时间只内
