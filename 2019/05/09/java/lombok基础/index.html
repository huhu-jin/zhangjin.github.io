lombok的基础使用

#简介
Lombok 是一种 Java™ 实用工具，可用来帮助开发人员消除 Java 的冗长，尤其是对于简单的 Java 对象（POJO）。它通过注解实现这一目的。

上图是 Lombok 处理流程，在Javac 解析成抽象语法树之后(AST), Lombok 根据自己的注解处理器，动态的修改 AST，增加新的节点(所谓代码)，最终通过分析和生成字节码。

@Data(重要)

这个注解相当于下面注解组合:

  • @ToString 生成toString方法
  • @EqualsAndHashCode 生成Equals和HashCode
  • @Getter 所有字段的Getter
  • @Setter on all non-final fields 非final字段get方法
  • @RequiredArgsConstructor final字段很nonnull字段生成构造器
